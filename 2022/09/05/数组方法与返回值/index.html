数组方法与返回值

2022/09/05 面试 共 2514 字，约 8 分钟

数组方法与返回值：

方法功能返回值是否修改原数组
arr instanceof Array检测数组boolean否
Array.isArray(arr)检测数组boolean否
arr.valueOf()返回数组的值{Object.prototype.valueOf()}该数组否
arr.toString()转换为字符串字符串，包含用逗号分隔的每个数组元素。否
arr.join([separator])转换为字符串，以分隔符连接元素。‘separator’缺失默认用“,”连接字符串否
arr.pop()删除最后一个元素。删除的元素是
arr.push(element1, …, elementN)将值追加到数组中。该数组的新长度。是
arr.shift()删除数组第一个元素。删除的元素是
arr.unshift(element1, …, elementN)将一个或多个元素添加到数组的开头。该数组的新长度是
arr.reverse()将数组中元素的位置颠倒新数组是
array.splice(start[, deleteCount[, item1[, item2[, …]]]])删除或替换现有元素或者原地添加新的元素来修改数组。arg2为0代表插入元素。[拼接]修改后的数组是
arr.concat(value1[, value2[, …[, valueN]]])合并两个或多个数组。（若无参数，返回调用此方法的现存数组的一个浅拷贝。对于新数组的任何操作（仅当元素不是对象引用时）都不会对原始数组产生影响。[元素都为基本类型时，独立、‘深拷贝’；元素存在引用类型，浅拷贝，会影响]）新数组否
Array.from(arrayLike[, mapFn[, thisArg]])Array.from() 方法有一个可选参数 mapFn`，让你可以在最后生成的数组上再执行一次map方法后再返回。相当于 Array.from(obj).map(mapFn, thisArg)元素存在引用类型，浅拷贝新数组否
arr.slice([begin[, end]])提取原数组中索引从 begin 到 end 的所有元素（包含 begin，但不包含 end）。‘-1’表示最后一个。[切片]，无参数可以拷贝数组，end被省略代表提取到数组末尾。所提取的数组否
arr.indexOf()验证数组中是否含有某个元素，返回第一个匹配到的元素在数组中所在的位置，如果没有，则返回 -1index||-1否
arr.lastIndexOf()验证数组中是否含有某个元素，从数组尾部开始查找，返回第一个匹配到的元素所在的位置，如果没有，则返回-1lastindex||-1否

数组高阶函数：

Array扩展：forEach、map、reduce、filter、sort 、every 、some 、find 、findIndex

方法功能返回值是否修改原数组
arr.forEach(callback(currentValue [, index [, array]])[, thisArg])对数组的每个元素执行一次给定的函数。用于：想遍历做一些操作。undefined不会=>数组内基本数据类型；会=>数组内引用数据类型。即改堆引用不改栈内存地址。
var new_array = arr.map(function callback(currentValue[, index[, array]]) { // Return element for new_array }[, thisArg])每个元素都按顺序调用一次 callback 函数，并将其返回。用于：想要新数组数据。新数组同上（意味着map可能改变原数组），如果元素有对象，则应当新建变量，避免改变原数组。
array.reduce(function(total, currentValue, currentIndex, arr), initialValue)函数作用在这个Array的[x1, x2, x3...]上，这个函数必须接收两个参数，结果继续和序列的下一个元素做累积计算。[x1, x2, x3, x4].reduce(f) = f(f(f(x1, x2), x3), x4)新数组否
var newArray = arr.filter(callback(element[, index[, array]])[, thisArg])函数依次作用于每个元素，然后根据返回值是true还是false决定保留还是丢弃该元素。新数组否
arr.sort([compareFunction(a, b)])如果没有指明 compareFunction ，那么元素会按照转换为的字符串的诸个字符的 Unicode 位点进行排序。（比较的数字会先被转换为字符串）。如果指明了 compareFunction ，那么数组会按照调用该函数的返回值排序。即 a 和 b 是两个将要被比较的元素：如果 compareFunction(a, b) 小于 0 ，那么 a 会被排列到 b 之前；compareFunction(a, b) 大于 0 ， b 会被排列到 a 之前。要比较数字而非字符串，比较函数可以简单的以 a 减 b，如下的函数将会将数组升序排列。新数组是
arr.every(callback(element[, index[, array]])[, thisArg])判断数组的所有元素是否满足测试条件。有假则假否
arr.some(callback(element[, index[, array]])[, thisArg])执行回调，直到找到一个使得 callback 返回一个“真值”（即可转换为布尔值 true 的值）。如果找到了这样一个值，some() 将会立即返回 true。否则，some() 返回 false。有真则真否
arr.find(callback(element[, index[, array]])[, thisArg])查找符合条件的第一个元素，如果找到了，返回这个元素，否则，返回undefined元素值||value否
arr.findindex(callback(element[, index[, array]])[, thisArg])查找符合条件的第一个元素，如果找到了，返回这个元素的索引，否则，返回-1index||-1否
